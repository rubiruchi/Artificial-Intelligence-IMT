In this challenge, we trained the agent by using the Double Q-Learning introduced by by Hando van Hasselt. For playing the game, We combined the game theory for improving the linear regression method which is used to estimate the W and bias of the Double Q-learning.

For staring the game, do the following steps:
+ put the save_rl file in the same directory of pyrat.py (pyrat/save_rl)
+ put tem.py in the AIs directory (AIs/tem.py)
